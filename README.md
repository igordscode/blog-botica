La revista digital de referencia de **Botica Guaraní**. Un ecosistema de contenido de alta autoridad científica enfocado en medicina de precisión, biohacking ético y longevidad en el mercado paraguayo.

## ✨ Características "Foda"
- **Diseño Editorial de Lujo:** Tipografía curada (`Playfair Display` & `Inter`) y grid asimétrico inspirado en publicaciones científicas premium.
- **Sofia AI Integration:** Módulos de "Sofia Insight" que conectan artículos con datos reales de activos y protocolos.
- **Performance de Élite:** Construido sobre Next.js 15 para carga instantánea y SEO imbatible.
- **Enfoque Clínico:** Áreas dedicadas a especialistas (Endocrinología, Nutrología, Performance).

## 🛠️ Stack Tecnológica
- **Framework:** [Next.js 15 (App Router)](https://nextjs.org/)
- **Estilos:** [Tailwind CSS](https://tailwindcss.com/)
- **Iconografía:** [Lucide React](https://lucide.dev/)
- **Despliegue:** [Vercel](https://vercel.com/)

## 🚀 Inicio Rápido

```bash
# Instalar dependencias
npm install

# Iniciar servidor de desarrollo
npm run dev
```

## 📂 Estructura de Carpetas
- `/src/app`: Rutas y lógica de páginas.
- `/src/app/blog/[slug]`: Template dinámico para lectura profunda.
- `/src/components`: Componentes modulares de UI.
- `/public`: Activos estáticos y logos.
